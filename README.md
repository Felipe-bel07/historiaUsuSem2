# Inventory Manager (Python)

This is a simple inventory management system developed in Python.  
It allows users to manage products through an interactive menu in the terminal.

---

## Features

- Add new products (name, price, quantity)
- Show current inventory
- Validate user input
- Interactive menu system
- Store products using lists and dictionaries

---

## Technologies Used

- Python 3
- Basic programming concepts:
  - Lists
  - Dictionaries
  - Loops (`while`, `for`)
  - Conditionals (`if`, `elif`, `else`)
  - Functions
